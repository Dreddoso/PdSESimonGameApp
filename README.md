##Stato del progetto
Allestimento in corso

##Informazioni Android
- Min SDK: API 24 (Android 7.0)
- Target SDK: API 34 (Android 14)
- Compile SDK: API 36

##Emulatore utilizzato per test
- Nome: "Medium Phone API 35" 
- Sistema operativo: Android 15
- Risoluzione: 1080 x 2400 
- Architettura: x86_64

##Come eseguire il progetto
1. Aprire il progetto in Android Studio
2. Sincronizzare Gradle
3. Avviare un emulatore o collegare un dispositivo reale
4. Premere il pulsante "Run"
